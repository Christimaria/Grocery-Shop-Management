# 🛒 Grocery Shop Management System

A desktop application to manage grocery store operations built with Python, Tkinter, and MySQL.

## Features
- **Workers** — Add, view, edit and delete employee records
- **Products** — Manage inventory with name, type, brand and price
- **Customers** — Store shopper contact details
- **Billing** — Generate itemised bills with auto-calculated totals

## Tech Stack
- Python 3
- Tkinter (GUI)
- MySQL (Database)
- mysql-connector-python

## Database Schema
5 tables — Worker, Product, Customer, Billing, BillingItem
with foreign key relationships between them.

## How to Run
1. Clone the repo
2. Install dependency:
   pip install mysql-connector-python
3. Set up MySQL and create the database
4. Update DB_CONFIG in app.py with your credentials
5. Run:
   python app.py
